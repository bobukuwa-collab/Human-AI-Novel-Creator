## 診断システム詳細

### 分析対象

人間が書いた文章のみを分析対象とする。AIが生成した文章は除外する。

### 各軸の測定基準

#### サイコパス度
- 感情語（喜び・悲しみ・怒り・恐怖）の出現頻度が低い
- 他者の感情を描写するシーンが少ない・淡白
- 暴力・死・痛みの描写に感情的な修飾がない
- 登場人物が目的のための手段として描かれる

#### 策士度
- 謀略・計略・伏線の描写が精緻
- 裏切り・欺き・操作のシーンを好む
- 「なぜそうするか」の計算が文章に滲む
- 不信・猜疑心の表現が多い

#### 自己愛度
- 一人称代名詞（私・僕・俺など）の使用頻度が高い
- 主人公が美化・理想化されている
- 承認・賞賛のシーンへの偏愛
- 他者よりも主人公の内面・外見への注目が多い

#### 共感力
- 感情語の豊富さと多様性
- 他者の視点に立った描写の頻度
- 「相手がどう感じるか」への言及
- 人物の内面描写の細かさ

#### 語彙知性
- 使用語彙の多様性（同じ言葉の繰り返しが少ない）
- 文の長さと構造の複雑さ
- 比喩・隠喩・象徴の使用
- 専門用語・難解語の適切な使用

### 作家タイプ名の決定ロジック

5軸スコアの組み合わせパターンから、以下のタイプ名を出力する（実装時に拡張可能）。

| パターン | タイプ名 |
|---------|---------|
| サイコパス↑ 策士↑ | 冷血な建築家 |
| サイコパス↑ 共感↓ | 無感覚の証人 |
| 策士↑ 自己愛↑ | 舞台裏の支配者 |
| 共感↑ 語彙知性↑ | 魂の代弁者 |
| 自己愛↑ 語彙知性↑ | 燃える詩人 |
| 共感↑ サイコパス↓ | 傷を知る語り部 |
| 語彙知性↑ 策士↑ | 精緻な観察者 |
| 全軸バランス型 | 沈黙の万華鏡 |

※ 境界値・複合パターンは実装時に詳細化する。

### ジャンル選択の診断への反映

選択したジャンルは最終的な分析文（テキスト）の生成時に考慮する。
スコア自体はジャンルに関係なく文章内容から計算する（ジャンルによるスコアの下駄は履かせない）。

## 著名作家アンケート仕様

### Phase 1 実装仕様

**表示タイミング**: 診断結果ページで「ホームに戻る」ボタンを押す直前に表示

**UI設計**:
- タイトル：「あなたが好きな作家を教えてください（任意・アンケート）」
- サブテキスト：「今後のサービス改善に活用します。診断結果には影響しません。」
- 入力方式：ジャンルフィルター付きオートコンプリート
- 「なし・回答しない」を常に最上部に表示
- 1回答につき1名（複数選択は Phase 2 以降に検討）

**データ保存先**: `author_survey_responses` テーブル（新規作成）
```sql
id           uuid primary key
user_id      uuid references users(id)
novel_id     uuid references novels(id)
author_name  text not null
created_at   timestamptz default now()
```

**バリデーション**: `AUTHOR_LIST.md` に登録された作家名のみ受け付ける

### Phase 2 実装条件・方針

- **着手条件**: 有効な作家回答が 500件 以上蓄積された時点
- **分析方針**: 「好きな作家X → 書いた文章のスコア分布」の相関を分析
- **出力**: 診断結果に「〇〇（作家名）に近い書き手の傾向」を追加
- **エンタメ割り切り**: Phase 2 は科学的根拠よりも「楽しさ・意外性」を優先する。精度より体験を重視。

## 複数セッション設計（ペンディング）

**状態**: 社内ミーティングで決定待ち

**選択肢と推奨**:

| 案 | 概要 | 推奨度 |
|----|------|--------|
| 案1 | 初回セッションのみ診断対象 | △ UX が制限的 |
| 案2 | 最新セッションの結果のみ表示 | ○ |

## ジャンルと診断の関係性

ジャンル選択そのものも性格データの一部として扱う。

| ジャンル選択傾向 | 分析文への反映例 |
|---------------|---------------|
| ホラー・ダークファンタジーを選ぶ | 「あなたは暗闇に引き寄せられる傾向があります」 |
| 哲学・孤独を選ぶ | 「あなたは内省的で、世界との距離を置いて観察する書き手です」 |
| 愛と恋・友情を選ぶ | 「人との繋がりを物語の核に置く書き手です」 |
| ランダムを選ぶ | 「ジャンルに縛られない自由な書き手の傾向が見えます」 |
