## 実装メモ

- このリストをそのままDBテーブル（`authors`）または JSON ファイルに変換してオートコンプリートに使用する
- 「ジャンル」はこのファイルのセクション名に対応する
- Phase 2 で各作家の文体特徴データを追加する列を設ける予定
- 追加・削除は PR レビュー後に反映する（恣意的な追加を防ぐため）
